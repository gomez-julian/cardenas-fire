Resumen

Propuesta de aplicación para Cárdenas Fire SAS, en la cual los usuario podrán realizar diversos Reportes que se manejan dentro del modelo de negocio.

Actualmente se ha implementado el informe fotográfico, con el cual se facilita la creación de Reportes que contienen evidencias fotográficas, por lo cual se facilita esta actividad.
Las funciones que se tienen hasta el momento son:

  • Crear nuevos reportes
  • Agregar evidencias (imágenes) con una descripción
  • En base a los ejemplos se acepta actualmente dos imágenes por cada descrición
  • Eliminar reportes o evidencias de un reporte
  • Generar el documento en tamaño A4 para imprimir con las evidencias

Una vez probada la aplicación, puede ponerse en contacto para más detalles acerca de los usuarios y futuras funciones.
